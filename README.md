# 🏠 Boston House Price Prediction Web App

This project predicts Boston house prices using a Machine Learning model.

## 🚀 Features

- Predict house prices using ML
- Flask web application
- Interactive web interface

## 🧠 Machine Learning Model

Algorithm used:
- Linear Regression

Libraries used:

- Python
- Scikit-learn
- Pandas
- NumPy
- Flask

## 📂 Project Structure

house-price-predictor

app.py  
templates/index.html  
static/style.css  
Boston House Price predict.ipynb  

## ▶️ Run Locally

Install dependencies

pip install flask scikit-learn numpy pandas

Run the application

python app.py

Open browser

http://127.0.0.1:5000
